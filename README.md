# HTML Bootcamp 🎮

An interactive, gamified HTML bootcamp designed to take you from basic HTML structure to advanced semantic concepts through 8 structured chapters and a live code editor.

## Features

- **8 Gamified Levels**: Lock/Unlock progression as you complete chapters.
- **Monaco Code Editor**: Fully integrated code editor (the same engine powering VS Code) with syntax highlighting.
- **Live Preview**: An iframe that renders your HTML code in real-time as you write it.
- **Auto-Grading Engine**: Parses and evaluates your HTML locally, giving you a score out of 100 and precise feedback on what to improve.
- **Local Persistence**: Saves your progress automatically in your browser's local storage.
